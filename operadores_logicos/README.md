<h1>Operadores Logicos</h1>
<p>Existen 3 operadores logicos el "and" el "or" y el "not" que sirven principalmente para ver si se cumple una cierta condicion si se cumple devuelve True por ejemplo "and" devuelve True si solamente se cumplen las dos condiciones que le pasamos si una no se cumple devuelve False. Con "or" es mas flexible solo necesita que una condicion se cumpla si es que una es falsa y otra verdadera devuelve True si ninguna se cumple devuelve False, mientras que "not" invierte los valores si es que algo era true con not ahora es False, basicamente dice "si esto es falso devuelve verdadero en la terminal"
