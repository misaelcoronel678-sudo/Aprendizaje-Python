<h1>Operadores Logicos</h1>
<p>Existen 3 operadores logicos el <strong>"and" el "or" y el "not"</strong> que sirven principalmente para ver si se cumple una cierta condicion si se cumple devuelve True por ejemplo "and" devuelve True <strong><em>si solamente se cumplen las dos condiciones</em></strong> que le pasamos si una no se cumple devuelve False. Con "or" es mas flexible solo necesita <strong><em>que una condicion se cumpla</em></strong> si es que una es falsa y otra verdadera devuelve True si ninguna se cumple devuelve False, mientras que "not" <strong><em>invierte los valores</em></strong> si es que algo era true con not ahora es False, basicamente dice "si esto es falso devuelve verdadero en la terminal"
